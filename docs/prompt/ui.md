# React 19 Actions 页面 UI 设计规范

## 🎯 核心架构
- **技术栈**: Next.js 16 + React 19 + Tailwind CSS 4
- **布局模式**: 垂直结构，顶部吸顶选择器，下方详细内容
- **背景色**: 灰白色调 (bg-gray-50)
- **最大宽度**: max-w-7xl，配合 px-4/6 边距

## 🎨 色彩系统
- **主色调**: 灰白色系背景 (bg-gray-50) + 白色卡片 (bg-white)
- **主要操作**: 蓝色 (#2563eb)
- **成功状态**: 绿色 (#16a34a)
- **警告/乐观更新**: 黄色 (#ca8a04)
- **错误状态**: 红色 (#dc2626)
- **特殊功能**: 紫色 (#9333ea)
- **文件相关**: 橙色 (#ea580c)

## 📐 布局特性

### 吸顶选择器
- **定位**: `sticky top-0 z-10 bg-white border-b`
- **设计**: 紧凑型按钮，只显示关键信息
- **内容**: 特性 图标 + 名称 + 难度标签
- **响应式**: 移动端垂直布局，桌面端水平布局

### 垂直内容区域
- **结构**: 3W 法则 → 特性 选择器 → 详细内容 → 官方示例
- **间距**: 统一使用 6 的倍数 (py-6, px-4/6, mb-4/6)
- **交互**: 点击选择器立即切换下方内容

## 🎭 组件设计

### 卡片系统
- **基础样式**: `border border-gray-200 bg-white shadow-sm`
- **圆角**: 大卡片 `rounded-lg`，小标签 `rounded-full`
- **阴影**: 普通 `shadow-sm`，悬停 `shadow-md`
- **选中状态**: `ring-2 ring-blue-500`

### 按钮设计
- **主要按钮**: 蓝色背景，hover 状态加深
- **禁用状态**: 灰色背景，cursor-not-allowed
- **加载状态**: 显示加载文字，禁用交互
- **过渡效果**: `transition-colors`

### 标签系统
- **结构**: `inline-flex items-center rounded-full px-2/3 py-1`
- **类型**: 类别标签、难度标签、状态标签
- **配色**: 语义化颜色，对应不同功能类型

## 📝 文字层级
- **主标题**: `font-bold text-3xl`
- **副标题**: `font-semibold text-xl`
- **小标题**: `font-semibold text-lg`
- **正文**: `text-gray-600`
- **代码**: `font-medium text-xs`

## 💻 代码展示
- **背景**: `bg-gray-900 text-gray-100`
- **容器**: `overflow-x-auto rounded-lg p-4`
- **复制功能**: 右上角复制按钮
- **语法高亮**: 保持代码可读性

## ⚡ 交互状态
- **Hover**: 颜色加深，transition-colors
- **选中**: ring 边框突出显示
- **Loading**: 占位符 + 加载动画
- **乐观更新**: 黄色背景标识临时状态

## 🔄 状态指示
- **成功**: 绿色背景 + checkmark 图标
- **错误**: 红色背景 + 错误图标
- **进行中**: 蓝色背景 + 旋转动画 

## 📱 响应式设计
- **移动端**: 单栏布局 `grid-cols-1`
- **桌面端**: 保持垂直布局，内容区域适配
- **容器**: `max-w-7xl mx-auto px-4/6`

## 🎯 关键要求
1. **吸顶选择器**: 紧凑设计，支持快速切换
2. **垂直布局**: 信息层次清晰，内容向下展开
3. **状态反馈**: 丰富的视觉状态指示
4. **交互演示**: 每个特性配备多个场景演示（用户核心要求）
5. **问题解决**: 针对性展示每个特性解决的具体问题

## 🎮 交互演示要求（用户核心需求）
- **多场景覆盖**: 每个特性至少3个不同应用场景的演示
- **真实交互**: 模拟网络延迟、状态变化、错误处理
- **教学价值**: 展示特性在不同情况下的使用方法和优势
- **视觉反馈**: 乐观更新、加载状态、成功/错误提示
- **实际应用**: 涵盖表单处理、社交功能、数据管理等常见场景

## 🚀 简化提示词
创建现代化React学习页面，采用垂直布局设计。顶部紧凑型吸顶选择器，下方展示详细内容。灰白色调背景，统一圆角卡片设计。**重点要求：每个特性必须配备多个不同场景的交互式演示，模拟真实使用情况**。包含代码示例、问题解决方案。风格专业清爽，注重用户体验和教学效果。
